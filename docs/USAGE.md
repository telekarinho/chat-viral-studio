# Manual de uso — Chat Viral Studio

> ⚖️ Tudo é **ficção/dramatização**. O selo "História fictícia para entretenimento"
> vem ligado por padrão.

## Os 4 modos

### 1. Modo IA Total
`Criar → IA Total` → escolha **categoria**, **duração** (15–90s), **intensidade**,
**emoção** e **final** → **Gerar história**. A IA devolve título, gancho, conversa,
narração, legenda, hashtags e gancho de parte 2. Você cai direto no **Editor**.

### 2. Modo Texto + IA
`Criar → Colar texto` → cole sua história → **Gerar**. O sistema divide em
mensagens naturais, cria personagens e ajusta pausas.

### 3. Modo Eu Narrando
`Narrar` → ligue/desligue a câmera, ajuste a velocidade do **teleprompter** →
**Gravar** (contagem 3·2·1) → narre enquanto o chat rola e o teleprompter desce.
**Pausar / Continuar / Finalizar / Refazer**. No fim, baixe o MP4.

### 4. Modo Reação com câmera
`Narrar?mode=reaction` (card "Reação c/ Câmera" no início) → sua câmera ocupa o
topo, a conversa rola embaixo. Também dá pra gravar só voz (desligue a câmera).

## Editor de chat
- **💬 Mensagens:** remetente, tipo (texto, 🎤 áudio fake, 🖼️ print, 📞 chamada
  perdida, 🚫 apagada, figurinha, sistema), hora, delay, emoção, "visto".
  Reordene (↑/↓), insira e remova.
- **🎨 Aparência:** 4 temas (Verde, Escuro, Business, Minimalista), cor/nome/lado
  dos personagens, status online, velocidade das mensagens, selo, legendas, marca.
- **🎙️ Voz:** escolha entre 8 vozes → **Gerar narração IA**. Cada mensagem vira um
  bloco de áudio sincronizado; a emoção ajusta pitch/velocidade.
- **📈 Viral Score:** nota 0–100 (gancho, curiosidade, retenção, emoção, final,
  parte 2) + sugestões de melhoria.
- **⬇️ Exportar:** veja abaixo.

## Painel de exportação
Escolha **formato** (1080×1920 / 720×1280 / 4K), **legendas**, **marca d'água**,
**selo de ficção**, **música de fundo**, **volumes**. Botões:
- **Exportar MP4** — grava no navegador e converte via FFmpeg.
- **Roteiro (.txt)**, **Legenda (.srt)**, **Thumbnail (.png)**, **Descrição (.txt)**.
- **Salvar projeto** (também há autosave a cada edição).

## Biblioteca
`Biblioteca` lista todos os projetos (título, status, duração, formato, data,
thumbnail). Abrir volta ao editor; lixeira apaga.

## Fluxos rápidos
1. **IA → MP4:** Criar → IA Total → Gerar → (editar) → Voz → Exportar MP4.
2. **Texto → MP4:** Criar → Colar texto → Gerar → Voz → Exportar.
3. **Sua voz:** Criar/abrir → Narrar → câmera + teleprompter → Gravar → Baixar.

## Dicas de viralização
- Gancho na 1ª mensagem (tensão imediata).
- Falas curtas, alternando os 2 lados.
- Sempre deixe um **gancho de parte 2** no final.
- Use **legendas** ligadas para retenção sem som.
